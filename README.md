# Premolider - Site Institucional

## Site institucional desenvolvido como parte de estudo e prática em desenvolvimento web front-end, para apresentar os produtos e soluções da empresa Premolider, especializada em fabricação de estruturas de concreto pré-moldado e inovação na construção de casas com placas de concreto.

O projeto foi desenvolvido com foco em performance, responsividade e experiência do usuário, utilizando a abordagem Mobile First.


Objetivo do Projeto:

- Criar um site institucional

- Apresentar os produtos da empresa

- Divulgar a nova solução de casas com placas de concreto

- Facilitar o contato com clientes

- Melhorar a presença digital da empresa


Produtos Apresentados:

- Muros Pré-Moldados

- Mourões de Concreto (reto e curvado)

- Placas de Concreto

- Treliças H8 e H12

- Malhas para construção

- Estruturas para laje

- Isopor para construção

- Sistema inovador de casas pré-moldadas em concreto


Tecnologias Utilizadas:

- HTML5

- CSS3

- JavaScript

- Mobile First Design

- Layout Responsivo

- Integração com WhatsApp


Responsividade:

O site foi desenvolvido utilizando a abordagem Mobile First, garantindo uma boa experiência em:

- Smartphones

- Tablets

- Desktops


Funcionalidades:

- Menu responsivo 
- Hero sections com chamadas de ação
- Página de produtos com cards informativos
- Formulário de contato integrado ao WhatsApp
- Botão de contato rápido
- Layout moderno para empresas de construção


Melhorias Futuras:

- Integração com backend para envio de formulários

- Sistema de orçamento online

- Galeria de projetos realizados

- SEO para melhor posicionamento no Google

- Painel administrativo para atualização de conteúdo
